# 🌿 Greenden

A simple plant store website built using HTML and Tailwind CSS.

## 🚀 Features

* Responsive landing page
* Product listing page
* Contact form page
* Clean and minimal UI

## 🛠️ Built With

* HTML5
* Tailwind CSS

## 📂 Pages

* Home (`index.html`)
* Products (`product.html`)
* Contact (`contact.html`)
